## [:beer: Brewery Memo :beer:](http://brewery-memo.work/)

##### クラフトビール醸造所の情報を共有するアプリ

http://brewery-memo.work/

## 🚀 機能

**醸造所情報 CRUD**

**投稿へのタグ付け**

**イイネ**

**フォロー**

**Google 認証ログイン**

**GoogleMap 埋め込み (投稿詳細ページ / タイトルで検索した結果を表示)**

**検索 (投稿タイトル・メモ・タグで検索可能)**
